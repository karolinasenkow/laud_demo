Recent advancements in bioinformatic and biocomputational tools have revolutionized the speed and quality of data collection, leading to larger repositories of clinical data than ever seen before. The rapidly growing volume of raw data worldwide calls for development of new resources that can allow for more efficient, guided, and standardized approaches to medical research. The Urinary Microbiome Database poses a step in this direction, providing an integrated workbench for researchers and clinicians in the urinary microbiome research community. This database embodies a streamlined framework for navigating and analyzing large quantities of urobiome data, including de-identified patient metadata, 16S rRNA sequence data, and whole genome sequence data.


Data and Database Query

The Urinary Microbiome Database is currently populated with metadata and genomic data from a study led by Dr. David Nelson of the Indiana University School of Medicine. The study aims to find the cause of urethritis in males. The metadata shows the participant ID, sample ID, sample event, microbial taxa type, microbial taxa, microbial count, and cure status of approximately 300 male patients. The user can run both advanced and simple SQL queries on this data by navigating to the "Database Query" tab.


Data Analysis

The user also has the option to perform data analysis using a variety of integrated bioinformatic and biostatistical tools. These tools can be found under the "Data Analysis" tab. For genomic data, the BLAST bioinformatics tool is embedded to compare an inputted query sequence with WGS and 16S sequences of random samples, as well as assembled pseudochromosomes of certain species. For metadata, several statistical tests are embedded to test significance of variable relationships. The user also has access to built-in supervised and unsupervised machine learning algorithms, including principal component analysis, k-NN classificiation, and random forest.
